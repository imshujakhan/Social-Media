# Social Media App

Full-stack social media application with authentication and CRUD operations.

## Features
- Login/Signup with authentication
- Create, Read, Update, Delete posts
- Like/Unlike posts
- Responsive design

## Tech Stack
- React 19 + React Router
- Bootstrap 5
- JSON Server
- Context API

## Setup

### Install
```bash
npm install
npm install -g json-server
```

### Run
```bash
npm run api
npm run dev
```

### Login
```
email: john@test.com
password: 123456
```

## API Endpoints
```
GET    /users
POST   /users
GET    /posts
POST   /posts
PUT    /posts/:id
DELETE /posts/:id
```
